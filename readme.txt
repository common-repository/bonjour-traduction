== Description ==

<strong>Bonjour Traduction</strong> affiche la traduction de "Bonjour" dans plus de 140 langues diff&eacute;rentes et al&eacute;atoirement dans votre panneau d'administration. Cette extension existe aussi en Anglais, <a href="http://www.maxence-blog.fr/2011/05/23/plugin-wordpress-hello-translate" target="_blank">cliquez-ici</a> pour en savoir plus.

Si une traduction que vous connaissez n'appara&icirc;t pas dans cette extension ou vous voulez apporter une modification, vous pouvez contacter l'auteur &agrave; cette adresse Email : <a href="mailto:contact@maxence-blog.fr" target="_blank">contact (at) maxence-blog (dot) fr</a>.

== Installation ==

Extraire le fichier zip et d&eacute;posez simplement le contenu dans le r&eacute;pertoire wp-content/plugins/ de votre installation de WordPress, puis activez le plugin depuis la page Plugins.
